# Installation

* Install [Metamod](https://cs2.poggu.me/metamod/installation/)
* Install [CounterStrikeSharp (CSSharp)](https://docs.cssharp.dev/guides/getting-started/). (**Note**: This step can be skipped if you install the [Querator with CSSharp release](https://github.com/hamzehlany-gif/Querator/releases/))
	* Go to this link: https://github.com/roflmuffin/CounterStrikeSharp/releases
	* Scroll down and download 'counterstrikesharp-with-runtime'
	* Extract the addons folder to the csgo/ directory of the dedicated server. The contents of your addons folder should contain both the counterstrikesharp folder and the metamod folder
	* Verify the installation by typing `meta list` on server console. You should see CounterStrikeSharp plugin by Roflmuffin
	* You can refer to https://docs.cssharp.dev/guides/getting-started for detailed instructions. Initially, it may seem a bit hectic, but trust me, it's worth it! :P 
* Install Querator 
	* Download the latest [Querator release](https://github.com/hamzehlany-gif/Querator/releases/) and extract the files to the csgo/ directory of the dedicated server.
	* Verify the installation by typing `css_plugins list` and you should see Querator by Lany listed there.

**Note**: If you want to use Querator on a Windows server, you will need the Windows build of `counterstrikesharp` which is available on its [releases page](https://github.com/roflmuffin/CounterStrikeSharp/releases)
